# ManuScriptAI

AI-powered historical manuscript recognition and translation system.

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Backend Setup

```bash
cd backend
python -m venv venv

# Windows
venv\Scripts\activate

pip install -r requirements.txt
python app.py
```

## Models

Model files are not included in this repository.

Download:
- model.safetensors
- language_rf.pkl

Place them in:

backend/model/
```
